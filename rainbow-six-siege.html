Rainbow Six Siege

Maîtrisez l’art de la destruction et les gadgets dans Tom Clancy's Rainbow Six Siege. Faites face à des combats rapprochés intenses, affrontements mortels, tactiques, jeu d’équipe et actions explosives à chaque moment. Découvrez une nouvelle ère d'échanges de tir féroces et une stratégie experte dans la lignée des anciens volets de la série Tom Clancy’s Rainbow Six.

CONFIGURATION REQUISE

MINIMALE :
Système d’exploitation : Windows 7, Windows 8.1, Windows 10 (64bit versions required)
Processeur : Intel Core i3 560 @ 3.3 GHz or AMD Phenom II X4 945 @ 3.0 GHz
Mémoire vive : 6 GB de mémoire
Graphiques : NVIDIA GeForce GTX 460 or AMD Radeon HD 5870 (DirectX-11 compliant with 1GB of VRAM)
Réseau : connexion internet haut débit
Espace disque : 61 GB d’espace disque disponible
Carte son : DirectX® 9.0c compatible sound card with latest drivers


RECOMMANDÉE :
Système d’exploitation : Windows 7 SP1, Windows 8.1, Windows 10 (64bit versions required)
Processeur : Intel Core i5-2500K @ 3.3 GHz or better or AMD FX-8120 @ 3.1 Ghz or better
Mémoire vive : 8 GB de mémoire
Graphiques : NVIDIA GeForce GTX 670 (or GTX 760 / GTX 960) or AMD Radeon HD 7970 (or R9 280x [2GB VRAM] / R9 380 / Fury X)
Réseau : connexion internet haut débit
Espace disque : 61 GB d’espace disque disponible
Carte son : DirectX® 9.0c compatible sound card 5.1 with latest drivers
