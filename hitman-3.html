Hitman 3

HITMAN 3 marque la conclusion dramatique de la trilogie du monde de l'assassinat. Devenez l'agent 47, un assassin légendaire, et redoublez de créativité et d'improvisation pour réaliser les exécutions les plus ingénieuses et spectaculaires qui soient, le tout dans des destinations bac à sable particulièrement dépaysantes.

CONFIGURATION REQUISE

MINIMALE :
Système d’exploitation et processeur 64 bits nécessaires
Système d’exploitation : OS 64-bit Windows 10
Processeur : Intel CPU Core i5-2500K 3.3GHz / AMD CPU Phenom II X4 940
Mémoire vive : 8 GB de mémoire
Graphiques : NVIDIA GeForce GTX 660 / Radeon HD 7870
DirectX : Version 12
Espace disque : 60 GB d’espace disque disponible


RECOMMANDÉE :
Système d’exploitation et processeur 64 bits nécessaires
Système d’exploitation : OS 64-bit Windows 10
Processeur : Intel CPU Core i7 4790 4 GHz
Mémoire vive : 16 GB de mémoire
Graphiques : Nvidia GPU GeForce GTX 1070 / AMD GPU Radeon RX Vega 56 8GB
DirectX : Version 12
Espace disque : 60 GB d’espace disque disponible
Notes supplémentaires : VR Minimal Requirements: CPU: Intel 6-core i7-8700 / AMD Ryzen 7 2700 GPU: NVIDIA RTX 2060 Super / AMD 5700XT
